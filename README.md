## 📁 Projects

### 1. 🏆 Final Project — Sales Dashboard (Group B2)
**File:** `MindX/MindX Final Project - B2 GROUP.pbix`

A collaborative group project that serves as the capstone of the MindX Power BI course. This dashboard provides a comprehensive overview of sales performance, helping stakeholders track key metrics and uncover business trends.

**Key features:**
- Sales KPIs (Total Revenue, Orders, Average Order Value)
- Sales trends over time (monthly / quarterly)
- Product and category performance breakdown
- Regional or customer segment analysis
- Interactive slicers for dynamic filtering
